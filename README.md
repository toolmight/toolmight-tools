## 🚀 About ToolMight

**ToolMight** is a collection of simple and fast developer tools designed to make everyday development tasks easier.

Instead of searching for different tools across the web, ToolMight provides them in one place with a clean interface and instant results.

No login required.
No clutter.
Just useful developer utilities.

---

## 🧰 Available Tools

Some of the tools currently available:

* JSON Formatter
* Regex Tester
* Base64 Encoder / Decoder
* URL Encoder / Decoder
* Hash Generator
* Text Utilities
* Encoding / Decoding Tools

More tools are continuously being added.
